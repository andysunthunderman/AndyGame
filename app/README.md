# 游戏集合 By Andy3re

这是一个基于 Nuxt 3 开发的现代化网页游戏集合项目，提供多个经典小游戏的在线体验。

## 🎮 游戏列表

### 已完成的游戏
1. **扫雷游戏** - 经典的逻辑推理游戏，支持标记模式和音效
2. **贪吃蛇** - 经典成长类游戏，使用方向键控制
3. **深海捕鱼** - 休闲娱乐游戏，鼠标控制捕鱼器
4. **打字挑战** - 提升打字速度和准确率的练习游戏

### 开发中的游戏
5. **多人坦克对战** - 实时对战游戏（占位符）
6. **飞机大战** - 经典射击游戏（占位符）

## ✨ 主要功能

- 🎨 **现代化UI设计** - 美观的渐变背景和玻璃质感效果
- 📱 **响应式布局** - 完美适配桌面和移动设备
- 🔍 **实时搜索** - 快速查找游戏
- ⏰ **实时时间显示** - 显示当前日期和时间
- 🌤️ **天气查询** - 查看当前天气信息
- 📖 **游戏说明书** - 详细的游戏操作指南
- 📊 **成绩统计** - 本地存储游戏成绩和统计
- 🎵 **音效支持** - 游戏音效和音量控制
- 🔧 **管理后台** - 查看游戏统计和管理数据

## 🚀 技术栈

- **前端框架**: Nuxt 3 + Vue 3
- **开发语言**: TypeScript
- **样式**: CSS3 + 响应式设计
- **数据存储**: LocalStorage
- **音效**: Web Audio API
- **图形渲染**: HTML5 Canvas

## 🎯 游戏操作指南

### 扫雷游戏
- **左键点击**: 揭开格子
- **右键点击**: 标记/取消标记地雷
- **标记模式**: 点击按钮切换标记模式
- **目标**: 找出所有非地雷格子，避免触碰地雷

### 贪吃蛇
- **方向键**: 控制蛇的移动方向
- **目标**: 吃食物让蛇成长，避免撞墙和撞到自己

### 深海捕鱼
- **鼠标移动**: 控制捕鱼器位置
- **按住左键**: 伸出钓鱼线
- **松开左键**: 收回钓鱼线
- **目标**: 在限时内捕获尽可能多的鱼

### 打字挑战
- **键盘输入**: 输入显示的词汇
- **回车键**: 确认输入
- **目标**: 在限时内正确输入尽可能多的词汇
